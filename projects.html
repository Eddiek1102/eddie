ECE 1100 Discovery Project

Snake Game

I created a replica of the popular snake game.

  • Code

Purpose

I originally planned to create a logic gates simulator with C++. Unfortunately, after completing the logic gates simulator, I found that it did not justify a semester's worth of learning. So, I decided to create a snake game with many of the skills that are taught in classes such as ECE 2036 at Georgia Tech. The final project for ECE 2036 is a snake/pacman game so I believed this was an appropriate project to demonstrate my learning.

C++ is a popular programming language used to develop operating systems, video games, and programming microcontrollers.

In order to learn the C++ programming language, I used notes and exercises from the MIT OpenCourseWare course as well as a 10 hour youtube tutorial by Caleb Curry.

Results

Below are a couple pictures of me playing the snake game.

Here are some takeaways from the project:

  • The quality of the final product was not as high as I had hoped for. I believe with more time, I would be able to learn to use the Unity/Unreal Engine that would allow me to make a snake game with higher resolution.
  • When the game starts, you are given a score of zero.
  • The 'E' represents the head of the snake and the 'f' represents the randomly placed fruit on the game board.
  • When the snake consumes fruit 'f' it graduately increases in length and the score updates.
  • The trail of 'o' represents the snake's body as it increases in length.
  • Here is another picture of the game when the snake consumes even more fruit.
  • Here, it shows the end of the game when the snake's head runs into its own tail.
  • When the game ends, the score stops updating and the game freezes.
